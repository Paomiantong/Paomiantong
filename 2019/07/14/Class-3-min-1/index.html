操作系统（英语：operating system，缩写作 OS）是管理计算机硬件与软件资源的计算机程序，同时也是计算机系统的内核与基石。操作系统需要处理如管理与配置内存、决定系统资源供需的优先次序、控制输入设备与输出设备、操作网络与管理文件系统等基本事务。操作系统也提供一个让用户与系统交互的操作界面。–引自baidu百科


[为什么操作系统这么难做]https://linux.cn/article-4545-1.html?pr

在科技发达的今天，相信大家多多少少都摸过电脑。
Windows操作系统，相信大家再熟悉不过，几乎天天都需要与其打交道。工作、生活、娱乐，等等几乎方方面面都已经离不开它。正是因为大家过于依赖微软的该系统，才导致了windows操作系统的市场越滚越大，以致没有第三方的操作系统能与其抗衡。对于美国来说，他们当然是乐意看到这样的局面。但是，对于其他国家或企业来说，肯定是不愿意一直这样走下去的，毕竟在核心技术上过于依赖他人，是会带来一定安全隐患的。“中兴事件”就是一个很好的说明。虽说我们现在一直在用着windows系统，但，同时我们也在进行国产操作系统的研发。像此前，在央视某期节目中上演的“国产操作系统与windows系统大比拼”的节目就说明了，相关部门已经在进行国产操作系统的研发。有趣的是在那次比拼中，国产操作系统的速度竟然更优于windows系统。不过对于那个结果，个人认为有待商榷，毕竟两个系统众多的参数都没有完全统一。不管怎样相关部门已经开始重视国产操作系统的开发。比如说像现在的中兴新支点、麒麟操作系统、深度国产操作系统，都是不错的桌面系统，也都能满足现在的日常工作所需。所以说，不管是华为的内部操作系统，还是现在正在开发的桌面国产操作系统，都是非常重要的。虽说，我们现在并不一定要马上去完全替代安卓或windows系统，但是，至少可以说，当我们准备好了就不怕别人突然来卡你的脖子。这就是开发国产操作系统的重要性。
